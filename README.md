Color your words! Similar words have Similar color.

## Usage

> Use spm@3.x

```
$ spm install word-color
```

> Use npm

```
$ npm install word-color
```

```js
var wordColor = require('word-color');

wordColor('words');         // rgb(188,174,18)
wordColor.rgb('words');     // [188,174,18]
```
